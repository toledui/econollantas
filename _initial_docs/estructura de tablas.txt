11) Relaciones resumidas (mapa mental rápido)

users ↔ roles (N:N)

roles ↔ permissions (N:N)

users → departments (N:1)

users → branches (N:1 principal) y branch_user (N:N multi)

courses → course_categories (N:1)

courses → lessons (1:N) → lesson_contents (1:N)

courses ↔ users (N:N) vía course_user (asignación efectiva)

lesson_progress (user+lesson) + lesson_video_progress (user+lesson)

course_progress (user+course resumen)

assessments → assessment_questions → assessment_options

assessment_attempts → assessment_answers

activity_events (log analítico)

library_resources → library_categories y resource_types

announcements (+ targets opcional)

scorecard_templates → scorecard_fields

scorecard_entries → scorecard_values

12) Índices y constraints recomendados (importante para dashboard)

course_user: índice (course_id, user_id) unique

course_progress: índice unique (course_id, user_id)
